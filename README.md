# Calculator Application

## Project Overview
This is a simple command-line Calculator Application developed using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Objective
To create a calculator that can perform basic mathematical operations based on user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu
- Input validation
- Error handling for division by zero

## Technologies Used
- Python

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Run the application:
   ```bash
   python calculator.py
   ```
